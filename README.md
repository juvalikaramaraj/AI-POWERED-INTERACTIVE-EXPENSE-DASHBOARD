# 💰 AI-Powered Interactive Expense Dashboard

An interactive Expense Dashboard developed using **Microsoft Excel**, **VBA**, and **AI-assisted prompt engineering** to analyze business expenses, monitor budgets, and support financial decision-making.

---

## 📌 Project Overview

This project provides an interactive dashboard for tracking and analyzing expenses across different departments. It enables users to monitor spending, compare budget vs actual expenses, and visualize financial performance using dynamic charts, KPIs, and filters.

The project also demonstrates how AI can accelerate VBA development while maintaining financial accuracy through testing and validation.

---

## 🎯 Project Objectives

- Monitor organizational expenses.
- Compare Budget vs Actual spending.
- Analyze department-wise expenses.
- Track monthly expense trends.
- Automate reporting using VBA.
- Improve financial decision-making through interactive dashboards.
